# Manual Testing Test Cases Repository

## Project Overview
This repository contains manual testing test cases created for web applications.  
It includes functional, regression, smoke, sanity, and exploratory testing scenarios with proper test documentation.

## Testing Types Covered
- Functional Testing
- Regression Testing
- Smoke Testing
- Sanity Testing
- Exploratory Testing
- Negative Testing
- UI Testing


## Test Case Structure
Each test case contains:
- Test Case ID
- Test Scenario
- Test Steps
- Test Data
- Expected Result
- Actual Result
- Status
